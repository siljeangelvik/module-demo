### Branch: Webpack

Initialize Node.js project  
`$ npm init -y`  
Install module bundler webpack   
`$ npm install webpack webpack-cli --save-dev`  
Install plugin html webpack   
`$ npm install --save-dev html-webpack-plugin`  
Run all build scripts  
`$ npm run build`
